# Power BI – Estudos Básicos

Repositório com estudos e projetos básicos desenvolvidos em Power BI.

## Ferramentas
- Power BI Desktop
- Excel / CSV

## Conteúdos abordados
- Importação de dados
- Tratamento básico de dados
- Criação de visualizações
- Medidas simples (SUM, COUNT, AVERAGE)
- Filtros e segmentações

## Objetivo
Demonstrar conhecimentos iniciais em Power BI e análise de dados.
